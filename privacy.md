# トケルよ プライバシーポリシー

最終更新日：2026年3月4日

## 収集する情報
本アプリは以下の情報を取り扱います。
- 問題の写真（撮影やカメラロール等から選択したもの）
- 選択した学年（AIが学年に合わせた説明をするために使用します。他の目的には使用しません。）
- 利用回数・利用履歴・用語集（iCloud 経由でお客様のデバイス間で同期に使用）

## 情報の利用目的
- AIによるヒント・解説の生成
- 利用回数の管理やデバイス間で同期

## 情報の送信先
- 問題の写真や画像はヒント・解説の生成のためOpenAIまたはAnthropicに送信されます
- 当方のサーバーには一切送信・保存されません
- OpenAIのプライバシーポリシー：https://openai.com/policies/privacy-policy
- Anthropicのプライバシーポリシー：https://www.anthropic.com/privacy （※開いたページから言語選択で日本語を選べます）
- Apple（iCloud同期・App Store決済）：https://www.apple.com/jp/privacy/

## 対象年齢
本アプリは13歳以上を対象としています。
13歳未満のお子様の情報を意図的に収集することはありません。

## Anthropic Usage Policy準拠
本アプリはAnthropic Usage Policyに準拠して運営しています。
https://www.anthropic.com/legal/usage-policy
